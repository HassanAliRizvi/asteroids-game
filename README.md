# Asteroids Game

A simple Asteroids game built in Python.

## How to Run

1) Clone the repository:

git clone <your-repo-url>
cd <your-repo-folder>

2) Make sure you have Python 3 installed.

3) Run the game using the terminal:

python3 main.py

## Dependencies

1) Python 3

2) Pygame (if applicable, mention the library)
